# CCHS Variable Ontology

Development folder for the CCHS variable ontology - semantic mapping of survey variables to standardised concepts.

## Purpose

Map CCHS variables across cycles to:
- Equivalent concepts (e.g., BMI variants → single BMI concept)
- Semantic hierarchies (smoking → tobacco use → substance use)
- Standard vocabularies (SNOMED-CT, LOINC, etc.)
- Cross-survey harmonisation (CCHS ↔ CHMS ↔ international)
