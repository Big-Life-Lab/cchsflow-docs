## Technical Notes

1. **VIEWSTATE Decoding**: ASP.NET stores form state in base64-encoded VIEWSTATE that contains binary data with NUL bytes. Must decode byte-by-byte, filtering NULs.

2. **renv Conflicts**: The project's renv targets R 4.2.x but current system has R 4.4.2. Use `Rscript --vanilla` with standalone scripts to avoid issues.

3. **ASP.NET Postbacks**: Level 2 scraping requires browser automation because the site uses JavaScript `__doPostBack()` for navigation - can't use simple HTTP requests.
