# Legacy PUMF documentation

This folder contains a subset of CCHS PUMF (Public Use Microdata File) documentation downloaded from Google Drive. **This collection is incomplete and stops at 2017-2018.**

## For complete documentation

Use these sources instead:

| Source | Coverage | Location |
|--------|----------|----------|
| OSF.io mirror | 2001-2023 | [`cchs-osf-docs/`](../cchs-osf-docs/) |
| Metadata catalog | All 1,262 files | [`data/catalog/cchs_catalog.yaml`](../data/catalog/cchs_catalog.yaml) |
| Curated collections | Selected files | [GitHub Releases](../../releases) |

## Why this folder exists

This was imported from Google Drive before the OSF.io synchronisation system was built. Files are retained for reference and are tracked in the main catalog with `source_namespace: gdrive_pumf_collection`.
